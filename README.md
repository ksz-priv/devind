
# Devind project

A project for learning how to manage a kubernetes cluster.


## Goals

- Create AKS

- Create namespaces: 
  - dev
  - stage
  - prod

- Connect to Azure DevOps

- Create pipeline for manage cluser.

- Public ip address + custom domain for prod environment.
